# Управление факультетами

## Описание проекта

Я объединила все задания связанные с факультетами и деканатом. Сделала в виде веб-приложения для управления факультетами университета. Позволяет создавать факультеты, добавлять к ним кафедры, преподавателей и сотрудников, а также сортировать информацию.

## Функциональность

- Добавление новых факультетов
- Добавление кафедр на факультет
- Добавление преподавателей (ФИО, кафедра, должность, стаж, дисциплины)
- Добавление сотрудников в состав факультета
- Просмотр полной информации о всех факультетах
- Сортировка факультетов по названию
- Сортировка факультетов по номеру корпуса

## Технологии

- Java 17 - основной язык программирования
- Spring Boot 3.5.11 - фреймворк для веб-приложений
- Spring MVC - для обработки запросов
- Thymeleaf - шаблонизатор для HTML
- Maven - сборка проекта
- HTML5/CSS3 - оформление страниц

## Требования

- Java 17 или выше
- Maven
